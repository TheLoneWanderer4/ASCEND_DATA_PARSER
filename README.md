# ASCEND_DATA_PARSER
Simple python script to parse data from a CSV file and graph it in matlab

Just call in the commandline

$ python proc.py data_file.csv 'Col1' 'Col2'

where:
data_file.csv == the csv file with your data 
and
Col1 / Col2 are the headers, name in the first row of the csv, for the two columns you want to graph, first will be x axis, second will be y axis
